# 有思 1.11.0 更新日志

发布日期：待定

## 版本定位

这是有思首次面向用户公开发布的版本，重点完善首次安装、训练记录、本地数据保存、Android APK 分发和 App Store 分发流程。

## 新增与调整

- 应用名称和主要界面统一为有思。
- 默认使用简体中文界面。
- 内置动作库精简为 5 个基础动作。
- 内置动作说明改为简体中文。
- 首次安装不再内置个人训练计划或个人训练记录。
- 清理历史品牌和无关发布痕迹。
- 用户页新增“关于有思（Yours）”，提供官网和 GitHub 仓库入口。
- 移除不可保存的本地档案编辑入口，避免误导。
- Android 本地恢复优化：可从 Documents/有思/backups 读取或选择备份包，支持恢复 iOS 端导出的备份。
- Android 公共文件夹暴露为 Documents/有思，便于用户手动放置备份包。
- 训练计时页备注改为按组独立保存，避免上一组备注自动带到下一组。
- 训练计时页右滑返回优化：跨动作返回时回到上一个动作的最后一组。
- Android 支持 release APK 分发。
- iPhone 版本已正式上架 App Store。

## 已知限制

- 暂未提供应用内自动更新功能。
- Android APK 需要用户手动下载安装新版。
- iPhone 版本通过 App Store 更新。

## 测试建议

- 新用户首次安装后，确认没有旧训练记录。
- 创建一个训练计划并完成一条训练记录。
- 退出应用后重新打开，确认记录仍然存在。
- Android 用户测试覆盖安装新版后数据是否保留。
