# 有思 1.11.1 Android

本次发布将 Android 版本同步到 iOS 最新主线能力，并补齐真机适配问题。

## 更新内容

- 支持计划归档、自由记录、双语界面和服务器同步协议 v2。
- 修复 Android 首次冷启动时系统栏边框短暂遮挡的问题。
- 优化英文界面长按钮在真机大字体环境下的显示。
- 完善 Android 公共 Documents 备份导出和 Yours Vault 导出。
- 覆盖安装旧版时保留本地数据、服务器地址和 API 配置。

## APK 选择

- 大多数 Android 手机：`Yours-android-arm64-v8a-1.11.1.apk`
- 较旧 Android 设备：`Yours-android-armeabi-v7a-1.11.1.apk`
- Android 模拟器或 x86_64 设备：`Yours-android-x86_64-1.11.1.apk`

请下载后核对 `SHA256SUMS.txt`。
