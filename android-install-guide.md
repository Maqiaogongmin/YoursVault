# 有思 Android 安装说明

## 下载

请从有思官网或 GitHub Releases 下载最新版 APK。

不要从陌生链接、转发文件或不明来源下载安装包。

## 安装

1. 下载 APK 文件。
2. 打开 APK。
3. 如果系统提示“禁止安装未知来源应用”，请按提示允许当前浏览器或文件管理器安装。
4. 安装完成后打开有思。

## 更新

如果已经安装旧版有思，可以下载新版 APK 后直接覆盖安装。

正常情况下，只要新版 APK 使用相同包名和相同签名，并且版本号更高，覆盖安装会保留本地训练数据。

## 注意事项

- 更新前建议备份重要训练数据。
- 不要卸载旧版后再安装新版，否则本地数据可能被删除。
- 如果系统提示“无法安装”或“签名不一致”，请不要继续操作，先联系开发者。
