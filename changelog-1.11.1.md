# 有思 1.11.1

## Android

- 同步 iOS 最新训练体验：计划归档、自由记录、双语界面和服务器同步协议 v2。
- 修复 Android 冷启动时系统栏边框短暂遮挡的问题。
- 优化英文长按钮在真机大字体环境下的显示。
- 完善 Android 公共 Documents 备份导出和 Yours Vault 导出行为。
- 继续保留旧版覆盖安装的数据、服务器地址和 API 配置。

## 同步

- 服务器同步继续使用 `protocolVersion: 2` 和 `identityMode: syncId`。
- 支持普通训练、自由记录、归档计划、删除状态和墓碑事件的跨端同步。

## 安装提醒

- Android 旧版用户可直接覆盖安装。
- 覆盖安装会保留本地训练数据和服务器同步配置。
- 不要先卸载旧版，否则本地数据会被系统删除。
