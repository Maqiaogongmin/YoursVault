# 有思 1.11.3

## Android

- 同步当前 1.11.3 主线体验，包含数据管理操作状态、备份诊断和同步流程改进。
- 保留 Android 系统栏、真机大字体按钮、公共 Documents 导出和 Yours Vault 行为。
- Android 继续隐藏 iCloud 入口，备份恢复使用文件选择器、分享面板、公共 Documents、Yours Vault 和自托管服务器同步。
- 覆盖安装旧版时保留本地训练数据、服务器地址和 API 配置。

## 同步与备份

- 服务器同步继续使用 `protocolVersion: 2` 和 `identityMode: syncId`。
- 继续支持普通训练、自由记录、归档计划、删除状态、墓碑事件和重复记录去重。
- 备份包可迁移服务器地址，但不迁移 API token、设备 ID 和同步游标。

## 安装提醒

- Android 旧版用户可直接覆盖安装。
- 更新前建议先在 App 内创建备份包。
- 不要先卸载旧版，否则本地数据会被系统删除。
