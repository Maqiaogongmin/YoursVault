# 有思 Yours 1.11.0 Beta

这是有思 Beta 的小范围测试版本。

## 下载

Android 用户请根据设备架构选择 APK：

- `Yours-android-arm64-v8a-1.11.0.apk`：大多数近年 Android 手机优先选择这个版本。
- `Yours-android-armeabi-v7a-1.11.0.apk`：较旧 Android 设备可尝试这个版本。
- `Yours-android-x86_64-1.11.0.apk`：主要用于模拟器或 x86_64 设备。

iPhone 用户后续请通过 TestFlight 链接安装。当前 TestFlight 外部测试仍在等待 Apple 后台状态修复，链接可用后会同步到官网和本 Release 页面。

## 本版说明

- 首次公开 Beta 测试版本。
- 默认提供干净的新用户状态。
- 内置简体中文动作内容。
- Android 本地恢复支持从系统 Documents/有思/backups 目录选择 iOS 备份包。
- 用户页新增“关于有思（Yours）”，提供官网和 GitHub 仓库入口。
- 移除不可保存的本地档案编辑入口，避免误导。
- 训练计时页备注改为按组独立保存；跨动作右滑返回时回到上一个动作的最后一组。
- 训练数据默认保存在设备本地。
- Android 版本不包含 iCloud 同步入口。

## 安装前提醒

- Android 首次安装如果提示“未知来源”，请按系统提示允许当前浏览器或文件管理器安装 APK。
- 如果你安装过旧测试包，且签名不同，可能需要先卸载旧版本再安装新版。
- 卸载 App 会删除本机训练数据，请先确认是否需要备份。

## 重要边界

本仓库当前只发布安装包和说明文件，不开放完整 App 源代码。
